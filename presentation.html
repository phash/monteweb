Technologie & Architektur

Backend

  • Java 21 + Spring Boot 3.4
  • Spring Modulith (modular)
  • PostgreSQL 16 + Redis 7
  • JWT-Authentifizierung
  • OIDC/SSO (optional)
  • WebSocket Echtzeit

Frontend

  • Vue 3.5 + TypeScript 5.9
  • PrimeVue 4.5 (Aura)
  • Pinia State Management
  • vue-i18n (DE + EN)
  • PWA mit Push Notifications
  • Vitest (87+ Tests)

Infrastruktur

  • Docker Compose
  • MinIO Dateispeicher (S3)
  • GitHub Actions CI/CD
  • Prometheus + Grafana
  • nginx Reverse Proxy
  • DSGVO-konform

Sicherheit

  • HSTS, CSP, X-Frame
  • Rate Limiting
  • DSGVO: Datenexport/-loeschung
  • Audit-Log
  • Kommunikationsregeln
  • End-to-End-Verschluesselung*

17 Module

  • Auth, Users, Family
  • School, Rooms, Feed
  • Messaging, Files
  • Jobboard, Cleaning
  • Kalender, Notifications
  • Admin

Self-Hosted

  • Eigener Server oder Cloud
  • Volle Datenkontrolle
  • Keine Abhaengigkeit
  • Open Source Basis
  • Multi-Tenant faehig
  • Einfaches Deployment

Weitere Highlights

📱

Progressive Web App

Installierbar ohne App Store. Responsive Bottom Navigation auf Mobilgeraeten, Sidebar auf Desktop.

🔔

Push Notifications

VAPID Web Push fuer Echtzeit-Benachrichtigungen bei neuen Nachrichten, Events und Diskussionen.

🌐

Mehrsprachig (DE/EN)

Vollstaendige Internationalisierung mit Browser-Locale-Detection und manueller Sprachumschaltung.

🔒

OIDC/SSO

Optionale Single-Sign-On-Anbindung ueber OAuth2/OIDC fuer bestehende Schulverwaltungs-Systeme.

📊

Monitoring

Prometheus-Metriken und Grafana-Dashboard fuer Systemueberwachung und Performance-Analyse.

📄

PDF-Export

Automatische Generierung von Stundenberichten und QR-Code-Blaettern als PDF-Download.

🛡️

DSGVO-konform

Datenexport, Account-Anonymisierung und konfigurierbare Datenhaltung gemaess EU-Datenschutz.

⚙️

Modulare Architektur

Jedes Feature einzeln aktivierbar. Spring Modulith sorgt fuer saubere Modul-Grenzen im Backend.
