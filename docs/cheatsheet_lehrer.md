# MonteWeb Cheat-Sheet: Lehrkraefte (TEACHER)

**Version 1.0 | Februar 2026**

---

## Wichtigste Funktionen

| Funktion | Wo zu finden | Beschreibung |
|----------|-------------|-------------|
| Beitraege erstellen | Dashboard oder Raum > Info-Board | Feed-Posts verfassen |
| Raum verwalten | Raeume > Raum > Einstellungen | Mitglieder, Beitrittspolitik, Chat |
| Diskussionen | Raum > Diskussionen | Threads erstellen und moderieren |
| Termine erstellen | Kalender > "Termin erstellen" | Raum- oder Bereichstermine |
| Formulare erstellen | Formulare > "Formular erstellen" | Umfragen/Einwilligungen |
| Jobs erstellen | Jobbboerse > "Job erstellen" | Elternstunden-Aufgaben |
| Dateien verwalten | Raum > Dateien | Ordner + Sichtbarkeit |
| Fotobox konfigurieren | Raum > Fotobox > Einstellungen | Berechtigungsstufen |
| Nachrichten | Seitenleiste > Nachrichten | Alle Rollen erreichbar |

## Top 5 Aktionen

| # | Aktion | Schritte |
|---|--------|----------|
| 1 | **Beitrag posten** | Dashboard > Titel + Inhalt > "Veroeffentlichen" |
| 2 | **Raum erstellen** | Raeume > Entdecken > "Raum erstellen" > Name + Typ + Bereich |
| 3 | **Familie zum Raum hinzufuegen** | Raum > Mitglieder > "Familie aufnehmen" > Familie waehlen |
| 4 | **Termin erstellen** | Kalender > "Termin erstellen" > Daten + Sichtbarkeit > Speichern |
| 5 | **Formular erstellen** | Formulare > "Erstellen" > Typ + Fragen + Bereich > Veroeffentlichen |

## Typische Workflows

### Raum einrichten
```
Raeume > Entdecken > "Raum erstellen" > Name + Typ "Klasse" + Bereich waehlen
> Einstellungen anpassen (Beitrittspolitik, Chat) > Familien aufnehmen
```

### Elternabend organisieren
```
Kalender > "Termin erstellen" > Sichtbarkeit "Raum" > Raum waehlen
> Datum/Uhrzeit > Optional: Job verknuepfen fuer Helfer
```

### Umfrage durchfuehren
```
Formulare > "Erstellen" > Titel + Typ "Umfrage" > Bereich/Raum waehlen
> Fragen hinzufuegen > "Veroeffentlichen" > Ergebnisse einsehen > CSV/PDF Export
```

### Job fuer Eltern erstellen
```
Jobbboerse > "Job erstellen" > Titel + Stunden + Max. Helfer
> Eltern bewerben sich > Stunden bestaetigen
```

## Raum-LEADER-Rechte

| Recht | Beschreibung |
|-------|-------------|
| Mitglieder verwalten | Hinzufuegen, entfernen, Rollen aendern |
| Beitrittsanfragen | Genehmigen / Ablehnen |
| Diskussions-Threads | Erstellen, archivieren, loeschen |
| Raumeinstellungen | Beitrittspolitik, Diskussionsmodus, Chat |
| Fotobox konfigurieren | Standardberechtigung festlegen |
| Raum archivieren | Raum fuer Nutzer unsichtbar machen |
| Dateien-Sichtbarkeit | Alle / Nur Eltern / Nur Schueler |
| Beitraege pinnen | Wichtige Posts anheften |

---

## Termin-Berechtigungen

| Ebene | Kann ich erstellen? |
|-------|:-------------------:|
| Raum-Termin | Ja (als LEADER des Raums) |
| Bereichs-Termin | Ja (immer) |
| Schul-Termin | Nein (nur SUPERADMIN) |

---

## Formular-Scopes

| Scope | Zielgruppe |
|-------|-----------|
| **Raum** | Alle Mitglieder eines Raums |
| **Bereich** | Alle Nutzer in einem/mehreren Schulbereich(en) |
| **Schule** | Alle Nutzer der Schule |
