# MonteWeb Cheat-Sheet: Bereichsleitung (SECTION_ADMIN)

**Version 1.0 | Februar 2026**

---

## Wichtigste Funktionen

| Funktion | Wo zu finden | Beschreibung |
|----------|-------------|-------------|
| Bereichsverwaltung | Seitenleiste > Bereichsverwaltung | Nutzer + Raeume im Bereich |
| Bereich waehlen | Bereichsverwaltung > Dropdown oben | Zwischen Bereichen wechseln |
| Nutzer verwalten | Bereichsverwaltung > Tab "Benutzer" | Nutzer suchen, Sonderrollen vergeben |
| Raeume verwalten | Bereichsverwaltung > Tab "Raeume" | Raeume erstellen, Mitglieder sehen |
| Sonderrollen | Bereichsverwaltung > Nutzer > Rollen | PUTZORGA + ELTERNBEIRAT vergeben |
| Raeume erstellen | Bereichsverwaltung > Tab "Raeume" | Neuen Raum im Bereich erstellen |
| Termine erstellen | Kalender > "Termin erstellen" | Bereichs-Termine |
| Formulare | Formulare > "Formular erstellen" | Umfragen fuer den Bereich |

---

## Top 5 Aktionen

| # | Aktion | Schritte |
|---|--------|----------|
| 1 | **Sonderrolle vergeben** | Bereichsverwaltung > Nutzer > PUTZORGA/ELTERNBEIRAT zuweisen |
| 2 | **Raum im Bereich erstellen** | Bereichsverwaltung > Raeume > "Raum erstellen" |
| 3 | **Nutzer im Bereich finden** | Bereichsverwaltung > Benutzer > Suche/Filter |
| 4 | **Bereichs-Termin erstellen** | Kalender > "Termin erstellen" > Sichtbarkeit "Bereich" |
| 5 | **Bereichs-Formular erstellen** | Formulare > "Erstellen" > Scope "Bereich" > Bereich waehlen |

---

## Typische Workflows

### Neues Schuljahr vorbereiten
```
Bereichsverwaltung > Bereich waehlen > Raeume-Tab
> Neue Raeume erstellen (Klassen) > Mitglieder zuweisen
```

### PutzOrga-Elternteil benennen
```
Bereichsverwaltung > Benutzer-Tab > Elternteil suchen
> Sonderrollen > "PUTZORGA" zuweisen
```

### Elternbeirat ernennen
```
Bereichsverwaltung > Benutzer-Tab > Elternteil suchen
> Sonderrollen > "ELTERNBEIRAT" zuweisen
```

---

## Sonderrollen-Uebersicht

| Sonderrolle | Wirkung |
|-------------|---------|
| **PUTZORGA** | Kann Putztermine fuer den Bereich konfigurieren |
| **ELTERNBEIRAT** | Erweiterte Rechte: Termine + Formulare auf Bereichsebene |

## Wichtig zu wissen

- Sie verwalten nur die Ihnen **explizit zugewiesenen** Bereiche
- Die Zuweisung erfolgt durch den SuperAdmin
- Sie koennen **keine Nutzer erstellen oder loeschen**
- Sie koennen Sonderrollen **nur fuer Ihren Bereich** vergeben
- Alle TEACHER-Rechte stehen Ihnen zusaetzlich zur Verfuegung
