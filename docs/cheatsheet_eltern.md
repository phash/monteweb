## Typische Workflows

### Elternstunden sammeln
```
Jobbboerse > Job waehlen > "Anmelden" > Arbeit erledigen > "Abschliessen"
> Lehrkraft bestaetigt > Stunden im Familienkonto
```

### Putztermin leisten
```
Putz-Orga > Termin waehlen > "Eintragen" > Am Putztag: QR scannen (Check-in)
> Putzen > QR scannen (Check-out) > Putzstunden im Familienkonto
```

### Familie einrichten
```
Familie > "Familienverbund erstellen" > Partner einladen ("Mitglied einladen"
> Name suchen > Rolle "Elternteil" > "Einladung senden")
```

## Hilfe bei Problemen

| Problem | Loesung |
|---------|--------|
| Kann mich nicht anmelden | "Passwort vergessen?" nutzen oder Admin kontaktieren |
| Menuepunkt fehlt | Modul evtl. deaktiviert -- Admin fragen |
| Kann Eltern nicht anschreiben | Eltern-Kommunikation gesperrt -- Admin fragen |
| Stunden nicht gutgeschrieben | Lehrkraft muss bestaetigen |
| Putz-Checkout vergessen | PutzOrga oder Admin kontaktieren |
