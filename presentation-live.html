Technologie & Architektur

Backend

  • Java 21 + Spring Boot 3.4
  • Spring Modulith (modular)
  • PostgreSQL 16 + Redis 7
  • JWT-Authentifizierung
  • OIDC/SSO (optional)
  • WebSocket Echtzeit

Frontend

  • Vue 3.5 + TypeScript 5.9
  • PrimeVue 4.5 (Aura)
  • Pinia State Management
  • vue-i18n (DE + EN)
  • PWA mit Push Notifications
  • Vitest (87+ Tests)

Infrastruktur

  • Docker Compose
  • MinIO Dateispeicher (S3)
  • GitHub Actions CI/CD
  • Prometheus + Grafana
  • nginx Reverse Proxy
  • DSGVO-konform

Sicherheit

  • HSTS, CSP, X-Frame
  • Rate Limiting
  • DSGVO: Datenexport/-loeschung
  • Audit-Log
  • Kommunikationsregeln
  • Flyway-Migrationen (V001-V026)

17 Module

  • Auth, Users, Family
  • School, Rooms, Feed
  • Messaging, Files
  • Jobboard, Cleaning
  • Kalender, Notifications
  • Admin

Self-Hosted

  • Eigener Server oder Cloud
  • Volle Datenkontrolle
  • Keine Abhaengigkeit
  • Open Source Basis
  • Multi-Tenant faehig
  • Einfaches Deployment
